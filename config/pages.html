Notice

Your game will be reviewed by the administrators before becoming visible to the public. This process could take 3-5 days, or even longer in some cases.

In order to pass the review, you must ensure that your submission meets the following requirements:

1. All information of the game are filled correctly.

2. The game is playable without depending on third-party resources.

3. The game is not duplicated with one that already exists on the site.

4. The game is not copyright-protected, or is explicitly allowed for redistribution.

We might also edit or delete certain information based on our needs.


HTML5 Games

The game file must be a valid ZIP file with a file named exactly `index.html` inside. The file will be used as the game's entry.

The game does not depend on any resources from any third-party domains. (Connecting to third-party hosts will be blocked by the website's CSP, which might result the game being rejected.)


Flash Games

The game file must be a valid SWF file.

The game must not contain a site lock, or any settings preventing it from running on other websites.

If the game depends on external files apart from the SWF, upload it in ZIP format as a HTML5 game with a tag 'flash'.


DOS Games

The game file must be a JS-DOS bundle. If you have a ZIP file, you can convert it into a JS-DOS bundle via the DOS Zone Studio.


Other Emulator Games

Other emulator-based games, such as N64, NES, SNES, are not currently supported to be added here. However you can play them with the game emulator under the 'apps' page.

Please test the game to see if it can be played when embedded. Games that are not playable or does not comply with the website's CSP will be rejected.

Once you have tested the game and confirmed that it functions correctly, click the button below again to upload.
